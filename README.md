# 🌽 Corn Disease Detection using Deep Learning

## 📌 Project Overview
This project is an AI-based web application that detects diseases in corn leaves using Deep Learning.  
Users can upload an image of a corn leaf, and the model predicts the disease type.

---

## 🚀 Features
- Upload corn leaf image
- AI predicts disease type
- Fast and accurate detection
- Simple user-friendly interface

---

## 🧠 Model Used
- Convolutional Neural Network (CNN)
- TensorFlow / Keras
- Image Classification
